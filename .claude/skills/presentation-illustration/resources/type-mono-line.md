# Type 1: Minimal Mono Line — 미니멀 모노 라인

> 검정 배경 위의 하얀 단선. 면(fill) 없이 선(stroke)만으로 형태를 표현한다.
> 가장 절제된 유형. Apple Keynote, Stripe 프레젠테이션 스타일에 가깝다.

---

## 핵심 원칙

- **선만으로 의미를 전달한다** — fill 사용 금지 (닫힌 경로여도 fill: none)
- **한 가지 굵기로 통일한다** — strokeWidth 편차 없음
- **여백이 형태를 정의한다** — 선을 더하기보다 빼는 방향으로

---

## 색상 규칙

| 역할 | 토큰 | 용도 |
|------|------|------|
| **주선** | `--vdl-100` | 핵심 형태를 구성하는 메인 선 |
| **보조선** | `--vdl-600` | 디테일, 내부 구조, 질감 |
| **가이드선** | `--vdl-800` | 점선 가이드, 그리드 보조선 (선택) |

- 최대 3단계 명도만 사용
- `--vdl-50`은 사용하지 않음 (너무 밝아서 주선과 차이 부족)
- fill 속성은 항상 `none`

---

## 스트로크 규격

| 파라미터 | 값 |
|---------|---|
| strokeWidth | 1.5 (기본) |
| strokeLinecap | round |
| strokeLinejoin | round |
| strokeDasharray | `4 6` (가이드선 전용) |

- 모든 주선/보조선은 동일한 1.5px
- 가이드선만 점선 (선택적 사용)
- 1.5px는 프레젠테이션 스케일에서 얇지만 선명한 균형점

---

## 메타포 사전

모노 라인은 2D 평면이므로 아이소메트릭과 다른 메타포 체계를 사용한다.

| 패턴 | 표현 방식 | 적합한 주제 |
|------|----------|-----------|
| **윤곽선** | 대상의 외곽선만 그림 | 도구, 디바이스, 오브젝트 |
| **다이어그램** | 노드 + 연결선 | 관계, 흐름, 네트워크 |
| **기하 추상** | 원, 선, 삼각형 조합 | 개념, 원칙, 철학 |
| **그리드/패턴** | 반복 요소의 배열 | 시스템, 규칙, 일관성 |
| **분해도** | 부분을 분리해 나열 | 구조, 구성요소, 분석 |
| **궤적/경로** | 곡선, 파동, 벡터 | 흐름, 진행, 변화 |

### 메타포 적용 예시

```
"사용자 여정" → 궤적/경로 — 좌→우로 흐르는 곡선 + 노드 포인트
"디자인 원칙" → 기하 추상 — 원 3개의 관계 (교집합 다이어그램)
"컴포넌트 구조" → 분해도 — 박스 윤곽을 레이어별로 분리
"일관된 시스템" → 그리드/패턴 — 동일 모듈의 정렬된 반복
```

## 구성 규칙

### 시각적 무게 중심

- 캔버스의 **광학 중심** (기하 중심보다 약간 위) 기준
- 좌우 대칭이 아니어도 되지만, 시각적 밸런스 필수

### 복잡도 제한

| 캔버스 크기 | 선분 수 상한 | 비고 |
|-----------|-----------|------|
| ~200px | 15~25 | 아이콘급 |
| ~400px | 30~50 | 중형 일러스트 |
| ~640px | 50~80 | 대형 일러스트 |

넘으면 단순화한다. 복잡도를 높이지 않는다.

### 네거티브 스페이스

- 캔버스의 40~60%는 비어있어야 한다
- 모노 라인은 여백이 많을수록 강해진다

## SVG 구조 예시

```jsx
<svg xmlns="http://www.w3.org/2000/svg"
     viewBox="0 0 400 400" fill="none">

  {/* 가이드선 (선택) */}
  <line x1="200" y1="80" x2="200" y2="320"
        stroke="var(--vdl-800)" strokeWidth="1.5"
        strokeDasharray="4 6" strokeLinecap="round" />

  {/* 주선 — 핵심 형태 */}
  <path d="..."
        stroke="var(--vdl-100)" strokeWidth="1.5"
        strokeLinecap="round" strokeLinejoin="round" />

  {/* 보조선 — 내부 디테일 */}
  <path d="..."
        stroke="var(--vdl-600)" strokeWidth="1.5"
        strokeLinecap="round" strokeLinejoin="round" />
</svg>
```

### 특징

- `<defs>` / `<filter>` 없음 — 모노 라인은 그림자를 사용하지 않음
- 모든 요소가 stroke 기반 — fill은 `none`만 허용
- 단순한 SVG 구조 = 작은 파일 크기
