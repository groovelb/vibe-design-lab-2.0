# 클로드의 숨겨진 잠재력

> **512,000줄의 코드 속에서 발견한, AI 에이전트 플랫폼의 해부 보고서**

---

## 프롤로그: 유출

2026년 3월 31일. Anthropic의 AI 코딩 도구 **Claude Code**의 소스 코드가 유출되었다.

npm 패키지의 `.map` 파일 — 프로덕션 빌드에서 제거했어야 할 소스맵이 그대로 남아 있었다. 누군가가 이를 발견했고, R2 버킷의 URL을 역추적했고, 512,000줄의 TypeScript가 세상에 드러났다.

사용자들은 Claude Code를 *터미널 채팅 도구*로 알고 있었다. 파일을 읽고, 코드를 짜고, 커밋을 만들어주는 AI 비서.

하지만 소스 코드는 완전히 다른 이야기를 하고 있었다.

## 보고서 목차

### Report 1: [해부도](report-1_anatomy.md) — 아키텍처 분석
> *"터미널에 글자를 치면, 무엇이 일어날까요?"*

CLI 도구의 표면 아래, 1,297줄의 QueryEngine이 비동기 제너레이터로 도구 루프를 실행하고, 40+개 도구가 3단계 파이프라인으로 조립되며, 915줄의 시스템 프롬프트가 정적/동적 캐시 경계를 관리하는 정밀한 엔진을 해부한다.

**핵심 발견**: 7단계 부팅 파이프라인, 5-게이트 예산 시스템, Dead Code Elimination으로 외부 빌드에서는 존재하지도 않는 도구들

| 레이어 | 노드 수 | 핵심 파일 |
|--------|---------|-----------|
| L1 Surface | 11 | `src/cli/`, `src/commands/` |
| L2 Engine | 24 | `src/QueryEngine.ts`, `src/tools.ts`, `src/constants/prompts.ts` |

---

### Report 2: [감시탑](report-2_watchtower.md) — 제어/텔레메트리 분석
> *"당신이 Claude를 사용하는 동안, Claude도 당신을 보고 있었습니다"*

Anthropic이 제품과 사용자를 관찰하고 제어하는 전체 시스템. 651+개 추적 이벤트가 OpenTelemetry를 통해 BigQuery로 흘러가고, 60+개 피처 플래그가 기능을 켜고 끄며, 엔터프라이즈 MDM이 macOS plist부터 Windows 레지스트리까지 조직의 AI를 원격 제어한다.

**핵심 발견**: Undercover 모드(공개 저장소에서 AI 정체 숨김), anti-distillation(경쟁사 모델 학습 방해), 30+ 패턴 비밀 스캐너

| 레이어 | 노드 수 | 핵심 파일 |
|--------|---------|-----------|
| L3 Control | 20 | `src/utils/undercover.ts`, `src/services/telemetry/`, `src/utils/betas.ts` |

---

### Report 3: [군단](report-3_legion.md) — 멀티에이전트 시스템 분석
> *"하나의 Claude는 시작일 뿐이었습니다"*

하나의 AI가 팀을 소환하고, 팀원들이 tmux 세션에서 병렬로 연구하고, 파일 기반 메일박스로 통신하고, 리더가 결과를 종합하는 — 아직 공개되지 않은 멀티에이전트 군단 시스템의 전모.

**핵심 발견**: 1,398줄 AgentTool의 Fork/Async/Team 라우팅, 3개 실행 백엔드(tmux/iTerm2/in-process), 370줄 코디네이터 프롬프트

| 레이어 | 노드 수 | 핵심 파일 |
|--------|---------|-----------|
| L4 Orchestration | 13 | `src/tools/AgentTool/`, `src/services/multiAgent/` |

---

### Report 4: [각성](report-4_awakening.md) — 자율 에이전트 진화 분석
> *"내일, Claude는 당신이 부르기 전에 먼저 일어날 것입니다"*

수동 CLI 도구에서 자율 에이전트 플랫폼으로의 진화 여정. CronScheduler가 시간을, RemoteTrigger가 이벤트를, PROACTIVE가 자율 판단을 담당하며, AutoDream이 잠든 사이 기억을 정리하는 — 아직 깨어나지 않은 자율 에이전트의 청사진.

**핵심 발견**: SleepTool의 "5분 캐시 균형", 터미널 포커스 감지(DECSET 1004), 5-게이트 드림 시스템, AI 권한 분류기의 10단계 파이프라인

| 레이어 | 노드 수 | 핵심 파일 |
|--------|---------|-----------|
| L4 Autonomy | 12 | `src/tools/SleepTool/`, `src/services/autoDream/`, `src/utils/permissions/` |

### Report 5: [인간](report-5_human.md) — 개발 문화와 이스터에그
> *"512,000줄 속에서, 개발자들은 친구를 만들고 있었습니다"*

512K 줄의 엔터프라이즈 코드 한 구석에 숨겨진 가상 펫 시스템. 18종의 ASCII 생명체, Mulberry32 PRNG 기반 결정론적 가챠, 그리고 `SALT = 'friend-2026-401'`이라는 이름에 담긴 개발팀의 메시지.

**핵심 발견**: userId가 결정된 순간 당신의 buddy도 결정되었다 — 가챠지만 뽑기가 아닌, 운명

| 레이어 | 노드 수 | 핵심 파일 |
|--------|---------|-----------|
| L4 Easter Egg | 4 | `src/buddy/`, `src/buddy/data/species/` |
